# Libft + GNL + Printf Project from School 19

> This project combines the first 3 projects of School 19.
```
    Libft
    GNL
    Printf
 ```

## How to compile the project
A simple 
```
make 
```
will compile all the project
### Libft
```
make libftstd
```
### GNL
```
make libftgnl
```
### Printf
```
make libftprintf
```
